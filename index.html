1
Install Homebrew

Homebrew is Mac's package manager — think of it as the App Store for developer tools. You'll only ever install it once, and it makes everything else in this guide possible. 

# Paste this into Terminal and press Enter
/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"
💡 What to expect This will open Terminal and ask for your Mac password. That's normal — it needs permission to install software. Type your password and press Enter (nothing will appear as you type — that's a security feature).
2
Install the Core Tools

Run each command below in Terminal, one at a time. Wait for each to finish before running the next — you'll know it's done when you see a $ prompt again.

⚡
Node.js The engine that runs OpenClaw — like the electricity that powers the whole system 
brew install node
🦊
OpenClaw Your AI assistant software — this installs the main program that runs your AI 
npm install -g openclaw
🔒
Cloudflared Your secure internet tunnel — lets your AI receive messages without exposing your Mac. Think of it as a secure post box. 
brew install cloudflared
📧
GOG — Gmail Connector This is how your AI reads and sends emails on your behalf — handles the secure connection to Google 
npm install -g gogcli
☁️
GitHub CLI Your AI's backup system — lets your AI save its memory and work to the cloud overnight 
brew install gh
3
Set Up OpenClaw

Run the setup wizard — it will ask you a few questions and create your AI's config file. It takes about 5 minutes. 

# This runs the setup wizard
openclaw setup
📋 What the wizard asks for The wizard will ask for your API keys — see below for where to get them. Have both ready before you run this command. Setup takes about 5 minutes.
🔑  Where to Get Your API Keys
Anthropic — The AI Brain
console.anthropic.com  → API Keys → Create Key
This is the key that gives your AI its intelligence. JC will set a spending limit on your key so there are no surprises.
Brave Search — Web Access
api.search.brave.com  → Subscribe → Free Tier
This lets your AI search the web. The free tier is plenty for daily use — no payment info needed.
4
Connect Your Gmail

This connects your Candosa Google account to your AI, so it can read your calendar and help manage your inbox.

Replace the email address with yours before running
# Replace 'yourname@candosa.com' with your actual email
GOG_KEYRING_PASSWORD=YourChosenPassword gog auth add yourname@candosa.com \
  --client jclaw-personal --services gmail,calendar,contacts
🌐 What happens next This opens a browser window asking you to log into your Candosa Google account. Click Allow — your AI will then be able to read your calendar and manage your inbox on your behalf.
🔐 About YourChosenPassword Replace YourChosenPassword with any password you choose. This is a local encryption password for storing your Gmail credentials — it never leaves your machine. Write it down somewhere safe.
5
Start Your AI

This starts your AI in the background. It will run quietly even when you close Terminal. Your AI is now live and listening. 

openclaw gateway start

Then verify it's running:

openclaw gateway status
✅ You're done! If you see "running" in the output — your AI is live. Everything is set up correctly. Welcome to the team 🦊
